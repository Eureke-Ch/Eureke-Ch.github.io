Python中如何通过字符串调用Python中的函数

2021-06-27

问题

  项目里设计了很多的函数，例如func_1,func_2,func_3等，当我们想要通过和项目名相同的字符串，去调用对应的函数时，有如下的四种方法：

1、eval()函数

  eval函数在Python中用于执行字符串表达式，并返回字符串表达式的值；

2、locals()函数

  locals()函数是Python的内置函数，通过它们可以一字典的方式访问局部变量

3、globals()函数

  globals()函数是Python的内置函数，通过它们可以一字典的方式访问全局变量

4、getattr()函数

  getattr()函数用于获取Python对象的属性值，如果将各个函数放入到同一个Class中，可以进行使用
