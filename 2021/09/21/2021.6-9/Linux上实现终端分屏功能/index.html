Linux上实现终端分屏功能

2021-09-21

问题

  linux上自带的bash shell不带分屏功能，且配色样式较为单调；

1、zsh 另外的Linux终端（oh-my-zsh)

  为了提升Linux终端的能力，zsh被开发出来；同时又为了简化终端的配置，增加主题，基于zsh开发了oh-my-zsh；

  • 可以使用命令cat /etc/shell 显示系统里存在的终端
  • 如果没有的话使用apt-get install zsh进行安装
  • 安装完成后需要切换默认的终端为知识：chsh -s /bin/zsh
  • 安装oh-my-sh：wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | sh
  • 主题选择：打开配置文件 ~/.zhsrc ，将 ZSH_THEME 值改为你所选的主题名称

2、分屏工具Tmux（多路复用器）

  Tmux 是会话与窗口的”解绑”工具，将它们彻底分离。类似的终端复用器还有 GNU Screen。但是Tmux更简洁，更好用！

  • （1）它允许在单个窗口中，同时访问多个会话。这对于同时运行多个命令行程序很有用。

  • （2） 它可以让新窗口”接入”已经存在的会话。

  • （3）它允许每个会话有多个连接窗口，因此可以多人实时共享会话。

  • （4）它还支持窗口任意的垂直和水平拆分。

安装Tmux：sudo apt-get install tmux
打开默认的终端，输入Tmux即可进入Tmux窗口，然后便可以使用了，具体的使用命令参考网址：https://www.ruanyifeng.com/blog/2019/10/tmux.html

以下是 Tmux 的最简操作流程

  • 新建会话tmux new -s my_session。
  • 在 Tmux 窗口运行所需的程序。
  • 按下快捷键Ctrl+b d将会话分离。
  • 下次使用时，重新连接到会话tmux attach-session -t my_session。

3、分屏工具Terminator（终端复用器）

  Terminator是一个强大的终端复用器，同时允许进行分屏操作，还可以根据自己的需求对终端的样式进行配置；

  • 安装：sudo apt-get install terminator
  • 安装完成后，打开终端输入terminator，即可进行terminator
  • 字体，颜色等配置的修改方法：可以自己新建～/.config/terminator/config文件或者自己鼠标放在terminator的任意空白处右击选择preferences随便设置一些就可以自动生成～/.config/terminator/config文件
